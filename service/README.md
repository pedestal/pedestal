Pedestal.Server
========================================

Developer Bootstrap
--------------------

You will need:

* Leiningen 2

Benchmarks
--------------------

    lein bench-log

This will run 1-2 minute benchmarks comparing com.clojure.log with
clojure.tools.logging.

Benchmarking code is in the `bench` directory and includes a Logback
configuration file.
